# 📈 Stock Market Management System (Tradzo)

## 🚀 About the Project

This project is a Stock Market Management System developed using Java. It simulates stock market operations such as managing stocks, tracking portfolios, and interacting through a structured menu-driven interface.

The system is designed with modular architecture, incorporating core stock functionalities, user interface handling, and database integration.

---

## 🛠 Tech Stack

### 💻 Programming Language

* Java

### 🧩 Concepts Used

* Object-Oriented Programming (OOP)
* Data Structures
* File Handling
* Menu-driven CLI Design

### 🗄 Database

* SQL (StockManagement.sql)

### 📊 Design & Documentation

* UML Class Diagram (PlantUML)

---

## ✨ Features

* 📊 Stock data management
* 📁 Portfolio handling

* 📂 File-based stock storage

## 📂 Project Structure

```
zNWatch-main/
│
├── Runner.java                # Main entry point

```

## ▶️ How to Run

1. Open project in any Java IDE (IntelliJ / Eclipse)
2. Compile all `.java` files
3. Run:

```
Runner.java
```
